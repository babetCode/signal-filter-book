*Kalman and Bayesian Filters in Python* is an introductory textbook on kalman and bayesian filters written using Jupyter Notebook by Roger Labbe. The recommended way to run the book is to downoad the notebooks from github, set up a conda environment and run a local jupyter server. At the time of his writing this was the best option for reactive notebooks. However, even <u><a href="https://github.com/rlabbe/Kalman-and-Bayesian-Filters-in-Python#downloading-and-running-the-book" target="_blank">Labbe wrote</a></u> that "This is admittedly a somewhat cumbersome interface to a book." Now, <u><a href="https://marimo.io/" target="_blank">marimo</a></u> and <u><a href="https://docs.astral.sh/uv/" target="_blank">uv</a></u> offer more modern ways to run and share reactive python notebooks with easy dependency management. My goal with this site is to adapt what I have learned from Labbe's work and make it more easily shareable with modern tools.
